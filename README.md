### The code:

Application to send and recieve messages to login users.

Devise API for the login

### Dependencies:

- Ruby on Rails
- RSpec is needed to run the feature and unit tests available in ./spec/.
- Mailboxer
- FactoryBot
- Devise
- Warden helper
- Postgres DB
- Pry
- Capybara
- ShouldaMatcher

### Setup:<br>

Fork repo.
Run bundler to install gems.

### Instructions:<br>

Start the application by

```irb
rails start
```
